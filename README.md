## 使用方法
0. 确保已经安装了 openjdk > 17
1. 转到 PyOS 所在的目录
2. 输入 `javac Main.java` 来编译
3. 输入 `java Main` 来启动

用户名：**root**
<br>
初始密码：**114514**
